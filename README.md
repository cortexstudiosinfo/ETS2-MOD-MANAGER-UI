## Espanol

Truck Manager nace con un objetivo principal: hacer que compartir el orden de mods de Euro Truck Simulator 2 y American Truck Simulator sea mucho mas sencillo.

En vez de explicar a otra persona que mods van primero, cuales van despues o tener que tocar archivos manualmente, puedes preparar tu orden de carga desde la app, guardarlo como preset y compartirlo de una forma mas clara.

La app esta pensada para jugadores que usan muchos mods y quieren organizar sus perfiles sin complicarse.

## Objetivo principal

- Gestionar el orden de carga de mods de ETS2 y ATS.
- Guardar configuraciones como presets.
- Compartir presets para que otros usuarios puedan usar el mismo orden.
- Evitar tocar archivos del perfil manualmente.
- Hacer que organizar mods sea mas rapido, visual y facil de entender.

## Que puedes hacer

- Elegir entre Euro Truck Simulator 2 y American Truck Simulator.
- Detectar perfiles locales y perfiles de Steam.
- Ver los mods instalados.
- Ordenar mods activos de forma visual.
- Guardar el orden en el perfil del juego.
- Crear, importar, renombrar y eliminar presets.
- Usar ajustes separados para ETS2 y ATS.

## Credenciales

Esta version publica no incluye credenciales privadas.

Las funciones locales funcionan sin credenciales. Si quieres usar funciones online como compartir o descargar presets desde la nube, necesitas configurar Firebase en tu copia privada.

Para conectarlo:

1. Crea un proyecto en Firebase.
2. Activa Firestore Database.
3. Crea una cuenta de servicio.
4. Descarga sus credenciales.
5. Renombra el archivo a `firebase_credentials.json`.
6. Ponlo junto a `main.py`.
7. No subas ese archivo a GitHub.

## English

Truck Manager was created with one main goal: to make sharing mod load orders for Euro Truck Simulator 2 and American Truck Simulator much easier.

Instead of explaining which mods go first, which ones go later or manually editing profile files, you can prepare your load order inside the app, save it as a preset and share it in a clearer way.

The app is made for players who use many mods and want to organize their profiles without making things complicated.

## Main Goal

- Manage ETS2 and ATS mod load order.
- Save configurations as presets.
- Share presets so other users can use the same order.
- Avoid editing profile files manually.
- Make mod organization faster, visual and easier to understand.

## What You Can Do

- Choose between Euro Truck Simulator 2 and American Truck Simulator.
- Detect local and Steam profiles.
- View installed mods.
- Sort active mods visually.
- Save the load order into the game profile.
- Create, import, rename and delete presets.
- Use separate settings for ETS2 and ATS.

## Credentials

This public version does not include private credentials.

Local features work without credentials. If you want to use online features such as sharing or downloading presets from the cloud, you need to configure Firebase in your private copy.

To connect it:

1. Create a Firebase project.
2. Enable Firestore Database.
3. Create a service account.
4. Download its credentials.
5. Rename the file to `firebase_credentials.json`.
6. Place it next to `main.py`.
7. Do not upload that file to GitHub.
